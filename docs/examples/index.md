# Examples

What `learnings.md` looks like in practice.

The LLM builds this over time. Each session adds discoveries. Knowledge persists.

---

## Get Started

Copy these files to your project and tell your agentic tool to use them:

```bash
curl -O https://raw.githubusercontent.com/Proteusiq/agentic/main/AGENTS.md
curl -o learnings.md https://raw.githubusercontent.com/Proteusiq/agentic/main/docs/examples/python.md
touch todo.md
```
